# HR-Analytics-Dashboard
# Project Objective
To develop an interactive HR Analytics Dashboard that enables organizations to monitor workforce metrics, analyze salary patterns, evaluate demographic distributions, and support data-driven HR decision-making through insightful visualizations and KPI reporting.
# Dataset Used
- <a href="https://github.com/saleem2411/HR-Analytics/blob/main/HR%20Analytics%20DB.csv">Dataset</a>
# KPIS
- How many employees are currently in the organization?
- What is the average employee salary?
- How does salary vary across employee demographics?
- Which cities have the highest salary expenditure?
- Who are the highest-paid employees?
- How do salary patterns differ by gender?

# Process
- Gathered HR employee data from Kaggle.com.
- Performed data cleaning by removing duplicates, handling missing values, and correcting inconsistencies using SQL Queries.
- Validated employee records to ensure data accuracy and consistency.
- Created a data model to support efficient reporting and analysis.
- Developed DAX measures for Total Employees and Average Salary.
- Defined key performance indicators (KPIs): Total Employees, Average Salary
- Implemented interactive slicers for: Age Range, City, Gender
- Built visualizations to analyze salary distribution across employee demographics.

# Project Insights
- The organization consists of 472 employees with an average salary of 58.86K.
- Male employees account for a higher share of total salary expenditure compared to female employees.
- Delhi and Mumbai show the highest salary expenditure among all cities.
- Salary trends fluctuate over time, suggesting changes in hiring activity, compensation adjustments, or workforce expansion.

# Final Conclusion
This HR Analytics Dashboard transforms employee data into actionable workforce insights by providing a clear view of salary distribution, employee demographics, and compensation trends. Through interactive filtering and KPI monitoring, the dashboard helps HR professionals identify workforce patterns, evaluate salary structures, and make informed decisions related to talent management, compensation planning, and organizational growth.
